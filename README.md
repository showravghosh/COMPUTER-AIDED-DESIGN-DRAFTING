# 🏗️ CAD Project Repository

This repository contains a complete collection of **Computer-Aided Design (CAD)** works, including practice drawings, assignments, experiments, components, and final project files. It is organized for academic and practical use, mainly focused on AutoCAD-based design.

---

## 📁 Project Structure

```
CAD/
│
├── Book_&_Manuals/
│   ├── AutoCAD reference materials
│   ├── Experiment documents (Exp 1–9)
│   └── Presentation slides
│
├── components (CAD)/
│   ├── Beds, dining tables, kitchen elements
│   ├── TV units and workspace designs
│   └── CAD blocks and legends
│
├── Mid/
│   ├── Assignments (Qs1–Qs4)
│   ├── Circuit design
│   ├── Experiments (EXP-1, EXP-4, EXP-5)
│
├── Final/
│   ├── Civil Plan (multiple versions)
│   ├── Shopping Mall design (OEL)
│   └── Performance project
│
├── Other/
│   ├── Practice drawings (1–12)
│   ├── Isometric projections
│   ├── Third angle projection
│   └── Miscellaneous CAD exercises
```

---

## 📌 Features

* ✅ Complete **AutoCAD project workflow**
* ✅ Includes **DWG source files** and **PNG previews**
* ✅ Covers:

  * Basic to advanced CAD practice
  * Engineering drawing techniques
  * Civil and architectural planning
* ✅ Organized by **Mid, Final, and Practice work**

---

## 🛠️ File Types

* `.dwg` → AutoCAD drawing files
* `.png` → Image previews of drawings
* `.docx` → Experiment documentation
* `.pdf` → Reference books/manuals
* `.pptx` → Presentation materials
* `.bak` → Backup CAD files

---

## 🎯 Purpose

This repository is useful for:

* 📚 Students learning **AutoCAD**
* 🏗️ Civil / Architecture beginners
* 🧑‍💻 Practice and assignment reference
* 📐 Engineering drawing improvement

---

## 🚀 How to Use

1. Open `.dwg` files using **AutoCAD** or compatible software.
2. Use `.png` files for quick preview.
3. Follow experiment documents for step-by-step learning.
4. Explore final projects for real-world design ideas.
